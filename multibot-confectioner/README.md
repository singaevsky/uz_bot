# AI-Помощник Кондитера

## Описание проекта
Мультиплатформенный чат-бот для кондитерской, принимающий заказы через Telegram, VK и Avito. Использует ИИ для общения с клиентами и генерации визуализаций тортов.

## Функциональность
- Прием заказов на торты/десерты с уточнением веса, ингредиентов, даты доставки
- AI-общение с персонализацией по возрасту/полу клиента
- Генерация визуалов тортов по описанию
- Единая база данных заказов
- Уведомления кондитеру через Telegram

## Технический стек
- Backend: FastAPI + Python 3.12 + asyncio
- Боты: aiogram 3.x, vk-api, Avito API
- AI: OpenAI GPT-4o-mini + DALL-E 3
- БД: Supabase (PostgreSQL)
- Хостинг: Docker + webhook

## Установка и запуск
1. Клонируйте репозиторий
2. Установите зависимости: `pip install -r requirements.txt`
3. Настройте переменные окружения в `.env`
4. Запустите приложение: `python main.py`
